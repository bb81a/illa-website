# 我们可以做什么

## 小部件的设计界面

数十个美丽的组件可以通过拖放和丢弃组件来轻松建立完美的交互式的
页面。

![image](https://cdn.illacloud.com/official-website/img/solutions/dashboard/design.svg)

## 连接到任意数据库或 APIs

无论数据库或API，您都用来管理数据。 您可以在 ILLA 中输入配置信息来快速
连接到您的数据源。
ILLA 将以安全的方式访问您的数据，确保您的
数据的安全。

![image](https://cdn.illacloud.com/official-website/img/solutions/cms/data-source.svg)

我们支持的数据源包括但不限于PostgreSQL、MySQL、
MongoDB、Snowflake、Amazon S3、Google Sheets、Airtable、Stripe等。

## 与 AI 代理人整合以分析数据

You can create an AI agent with data analysis or question-answering capabilities
and seamlessly integrate it into your dashboard. This AI agent can assist you in
analyzing data by providing insights and answering questions.

![image](https://cdn.illacloud.com/official-website/img/solutions/dashboard/analysis.gif)
